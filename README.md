# רישום "נתתי לשליח" דרך הטלפון → המערכת שלך

שרת קטן שמחבר את **ימות המשיח** למערכת ההזמנות שלך.
מתקשרים, מקישים סכום, והוא נרשם אוטומטית במסך **"חשבון שליח"** — בלי לפתוח מחשב.

```
טלפון → מערכת ימות → השרת הזה → Firestore (workspaces/037220172) → האפליקציה מתעדכנת לבד
```

---

## מה צריך להכין

1. **Node.js 18 ומעלה** מותקן.
2. **מפתח Service Account של Firebase** (קובץ `serviceAccount.json`):
   - היכנס ל‑[Firebase Console](https://console.firebase.google.com/) → הפרויקט `jhauc-fd956`.
   - ⚙️ **Project settings** → לשונית **Service accounts** → **Generate new private key**.
   - שמור את הקובץ שירד בשם `serviceAccount.json` **באותה תיקייה** של `index.js`.
   - ⚠️ הקובץ הזה סודי — אל תשתף אותו ואל תעלה אותו לאינטרנט בפומבי.
3. **חשבון בימות המשיח** עם יחידות/דקות לשימוש ב‑API.

---

## הרצה מקומית (לבדיקה)

```bash
npm install
# מומלץ להגדיר קוד גישה כדי שלא כל אחד יוכל לרשום:
PIN=1234 npm start
```

השרת יעלה על פורט 3000. בדיקה: פתח בדפדפן `http://localhost:3000/health` — אמור להופיע `OK`.

---

## העלאה לאוויר (כדי שימות תוכל להגיע אליו)

ימות צריכה **כתובת HTTPS ציבורית**. אפשרויות נפוצות:
- **Render / Railway** — חינמי להתחלה, מתאים מאוד.
- **Firebase Cloud Functions** — נשאר באותו פרויקט (jhauc-fd956).
- שרת VPS משלך.
- לבדיקה זמנית בלבד: כלי tunnel כמו `ngrok`.

משתני סביבה שכדאי להגדיר בענן:

| משתנה | ברירת מחדל | למה |
|---|---|---|
| `PIN` | (ריק) | קוד גישה טלפוני — **מומלץ מאוד למלא** |
| `WORKSPACE_CODE` | `037220172` | קוד החיבור של המערכת שלך |
| `ENTRY_DESC` | `נרשם בטלפון` | תיאור שמסמן שהרשומה הגיעה מהטלפון |

## חיבור בצד ימות המשיח

1. בממשק ניהול המערכת בימות, צור/בחר שלוחה.
2. הגדר את סוג השלוחה כ‑**API** (קבלת נתונים משרת חיצוני).
3. בשדה כתובת ה‑API הזן את כתובת השרת שלך, למשל:
   `https://your-app.onrender.com/`
4. התקשר למערכת, היכנס לשלוחה — ותתחיל הזרימה: קוד גישה (אם הגדרת) → סכום → אישור → נרשם.

---

## איך זה נראה למתקשר

1. *"נא הקש את קוד הגישה"* (אם הוגדר PIN)
2. *"נא הקש את הסכום שנתת לשליח, בשקלים שלמים"*
3. *"הקשת 250 שקלים. לאישור הקש 1, לביטול הקש 2"*
4. *"נרשם בהצלחה. נתת לשליח 250 שקלים. להתראות."*

הרשומה תופיע מיד במסך "חשבון שליח", בראש הרשימה, עם התיאור "נרשם בטלפון".

## הערות

- **רק שקלים שלמים** נתמכים כרגע (הכי אמין בטלפון). אם תרצה אגורות — אפשר להוסיף הקשת ספרות נוספות אחרי נקודה.
- אין צורך בהקלדת טקסט — רק מספרים, אז זה עובד חלק.
- העלות: ימות גובה לפי יחידות/דקות שיחה; ל‑Firebase יש מכסה חינמית נדיבה לשימוש כזה.
